SongLister V1.2.1-6

SongLister is a C project designed for managing and displaying music playlists. It provides functionality for sorting songs based on ratings, searching for songs with specific ratings, and generating organized music file names. The project includes features for efficient memory allocation, file handling, and various string manipulations. With a focus on user-friendly music management, SongLister offers a comprehensive set of tools for organizing and presenting song information within a console environment.
